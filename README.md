Este projeto é uma plataforma de vídeos que oferece recursos de compra, venda e aluguel de filmes. Aqui, os usuários poderão explorar um amplo catálogo de filmes, realizar transações para comprar ou alugar obras, além de poderem vender filmes que possuam em formato digital. Além disso, a plataforma oferece um sistema de streaming de vídeo de alta qualidade e recomendações personalizadas com base nos interesses dos usuários. No repositório, os usuários encontrarão a documentação relacionada aos requisitos do projeto, a arquitetura do sistema, o plano de testes e outras informações relevantes para o desenvolvimento e utilização da plataforma.
